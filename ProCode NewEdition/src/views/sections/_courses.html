Курсы

Boot-camp

2 месяца

Программа:
  • - Азы верстки (HTML/CSS)
  • - Адаптивная верстка (верстка под мобильные устройства)
  • - Bootstrap (CSS-фреймворк)
  • - GULP/SASS

2200 грн/мес.

JavaScript Start

1.5 месяца

Программа:
  • - Базовые операции в JS.
    (условия, циклы, типы данных и прочие базовые понятия)
  • - Асинхронность
    (Асинхронность – это то, что делает JavaScript сложным даже для практикующих программистов. Этот блок дает глубокое понимание асинхронности. Без этих знаний на JavaScript вы будете писать только наугад. Абсолютно необходимый навык будущему фронт-ендщику.)

2800 грн/мес.

JavaScript Front-end

~6 месяцев

Программа:
  • - Работа с DOM
  • - ООП в JavaScript
  • - Promise/Generator
  • - AJAX, WebSocket, а также другие форматы передачи данных (JSON и прочие)
  • - Хранилища
    (Cookie, Local Store, WebSQL)
  • - HTML 5 API
    (Geo, D&D, File, History, Audio/Video и прочие)
  • - Canvas
  • - Angular.js
  • - Стажировка в реальной студии

3600 грн/мес.
